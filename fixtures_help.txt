При обычном создании:
python manage.py dumpdata products.Products > goods.json
python manage.py dumpdata products.ProductCategory > categories.json

потом при загрузке была ошибка:
django.db.utils.IntegrityError: Problem installing fixture

Решение -->

Создание базы данных

python manage.py dumpdata --exclude=contenttypes --exclude=auth.Pe
rmission products.ProductCategory > categories.json

python manage.py dumpdata --exclude=contenttypes --exclude=auth.Pe
rmission products.Product > goods1.json


Далее вручную поменял иероглифы и загрузил

python manage.py loaddata products/fixtures/categories.json

python manage.py loaddata products/fixtures/goods.json
